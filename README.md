# CV Pro Pack – Gumroad Project

Ce dépôt contient tous les fichiers utilisés pour créer et vendre le **CV Pro Pack – 12 Templates Premium (PDF + Word)** sur Gumroad.

## Structure du dépôt

- `templates-pdf/` – PDFs finaux des CV (originaux + variations couleurs)
- `templates-docx/` – Versions modifiables (Word/Google Docs)
- `guide/` – Guide PDF et textes sources
- `prompts/` – Prompts ChatGPT pour rédiger le CV
- `gumroad/` – Texte de fiche produit, tags, idées d’upsell
- `assets/` – Covers, mockups et images de preview pour Gumroad
- `workflow/` – Process 15 minutes pour (re)générer le pack

## Source open‑source

Les templates originaux proviennent de :
- Repo : `jankapunkt/latexcv`
- URL : https://github.com/jankapunkt/latexcv
- Licence : MIT (réutilisation commerciale autorisée)

Les fichiers de ce dépôt sont une **version modifiée, rebrandée et organisée** pour être utilisée comme produit digital sur Gumroad.
